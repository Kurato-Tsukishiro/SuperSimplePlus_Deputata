## 機能一覧
#### ホストのみ
#####　キーボードショートカット
| キー                    | 機能           | 使えるとき     |
| ----------------------- | -------------- | -------------- |
| `左Shift`+`A`+`右Shift` | 廃村           | 試合中いつでも |
| `左Shift`+`C`+`右Shift` | 投票&開票をスキップ | 会議中         |
##### 設定
|機能                |
|PC以外をキックする|
|PC以外をバンする|
#### その他機能
| 機能             | 操作                                                       |
| ---------------- | ---------------------------------------------------------- |
| アップデート機能 | タイトル画面の右上のアップデートボタンを押してアップデート |
